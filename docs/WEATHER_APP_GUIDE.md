WEATHER APP GUIDE 🌤️

## Introduction

This guide explains the structure and functionality of the Weather App built with SwiftUI. It shows how the app works and how to run it on your device or simulator.

## Prerequisites

 To run the Weather App, you need:
Xcode 14 (or newer) installed on macOS
A Mac capable of running SwiftUI projects
Optional: an iPhone or iPad to test on a real device

## Setting Up the Project

Clone the repository:
git clone https://github.com/mariamtabagari27/WeatherApp.git
Open the project in Xcode by double-clicking the .xcodeproj or .xcworkspace file.
Make sure your target device is set (simulator or your iPhone).

## Running the Application

Press the Run button (▶️) in Xcode, or use Cmd + R.
The app will launch on the selected simulator or device.

Explore the interface:
- View the main temperature for the first day
- See the 5-day weather forecast
- Tap "Change Day Time" to toggle between day and night
  
## Features

- Dynamic daily weather (randomized temperatures for now)
- 5-day forecast with icons
- Day/Night toggle with smooth animation
- Clean and responsive SwiftUI interface
  
## API Integration (Future)

Currently, the app uses randomized temperatures for demonstration.
In the future, it can integrate with a real API, such as OpenWeatherMap, to fetch real-time weather.

## Troubleshooting

If the app doesn’t run:
- Make sure you are using Xcode 14 or newer
- Ensure your Mac meets SwiftUI requirements
- Clean the build folder (Cmd + Shift + K) and rebuild
-Check that the simulator/device is properly connected.

## Conclusion :)

This guide provides everything you need to run and explore the Weather App. You can expand it later by connecting it to a real weather API or customizing the interface. <3
