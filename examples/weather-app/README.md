# Weather App

## Overview
This Weather App provides real-time weather data for any location worldwide. Users can search for any city and view detailed weather information such as temperature, humidity, wind speed, and forecasts.

## Features
- Real-time weather updates
- Search by city name
- Display of current weather conditions
- 5-day weather forecast

## Technologies Used
- HTML
- CSS
- JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mariamtabagari27/desktop-tutorial.git
   ```
2. Navigate to the project directory:
   ```bash
   cd desktop-tutorial/examples/weather-app
   ```
3. Open index.html in your browser to view the app.

## Usage
- Input the name of the city in the search bar.
- Click the search button to fetch the weather details.
